### High Stakes Turing Test Protocol

**Can you distinguish the signal from the noise?**
Reality Wager is a gamified "Turing Test" for visual media. Players are presented with an image and must wager their virtual credits on whether the subject is **REAL** (Human/Photography) or **AI** (Generated).

## âš¡ Features

* **High Stakes Betting:** Wager 5%, 20%, or 50% of your net worth on every decision.
* **The Back Room:** Go bankrupt (<$10)? You get sent to a black terminal where you must solve manual labor tasks (math problems) to earn your buy-in back.
* **Cyberpunk Aesthetic:** Custom CRT scanlines, neon glows, and glitch effects using Tailwind CSS.
* **Supabase Integration:** Real-time database for global user balances and game history (optional).
* **Fail-Safe Protocol:** Includes an "Anonymous Login" system that auto-funds new players with $1,000 credits.
